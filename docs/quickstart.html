Quick Start

Three ways to access the data. Pick whichever fits how you work.

Option 1 — Ask an AI (no code)

Connect any MCP-compatible AI assistant (Claude Desktop, Claude Code, Cursor, OpenCode, etc.) to the live SSWR database. No SQL, no downloads — ask questions in plain language.

New to MCP? Give this prompt to your AI assistant

Click the button below to copy the setup prompt, then paste it into Claude Code, Cursor, or any MCP-capable tool. The AI will find your config file, add the connection, and walk you through the rest.

Option 2 — Python or R

Download the CSV files and start analyzing locally.

Python

import pandas as pd

papers  = pd.read_csv("https://github.com/beperron/SSWR-History-Public/raw/main/data/csv/papers.csv")
authors = pd.read_csv("https://github.com/beperron/SSWR-History-Public/raw/main/data/csv/paper_authors.csv")

# Methodology distribution
papers.methodology.value_counts()

# Presentations per year
papers.groupby("year").size()

# First authors from a specific institution
first = authors[authors.author_order == 1]
first[first.institution_normalized == "University of Michigan"]

R

library(readr)

papers  <- read_csv("https://github.com/beperron/SSWR-History-Public/raw/main/data/csv/papers.csv")
authors <- read_csv("https://github.com/beperron/SSWR-History-Public/raw/main/data/csv/paper_authors.csv")

# Methodology distribution
table(papers$methodology)

# Join and analyze
merged <- merge(authors, papers, by.x = "paper_id", by.y = "id")
table(merged$year, merged$methodology)
