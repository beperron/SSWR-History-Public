Methodology Classifications & Taxonomy

This page documents the classification systems used in the SSWR Conference History Database, including research methodology categories, academic position taxonomy, and presentation format definitions. For a full description of the classification process, see the companion paper.

Research Methodology Classification

Each presentation’s methodology was classified based on analysis of its abstract content using a small language model (gpt-oss-20b). The classification prompt instructed the model to analyze each abstract for methodological indicators including data type descriptions, sample characteristics, analytical methods mentioned, and explicit methodology statements.

Validation: A random stratified sample of 60 abstracts (10 per category) was manually classified to calculate inter-rater agreement between human and model classifications. Cohen’s kappa was .83, indicating almost perfect agreement. Accuracy varied by category: quantitative (100%), qualitative (100%), review (90%), mixed methods (80%), and theoretical/other (75%).

Category Definitions

Category Count % Definition
quantitative 14,542 61.1% Studies employing numerical data collection and statistical analysis methods, including surveys, administrative data analysis, experimental and quasi-experimental designs, and secondary data analysis.
qualitative 5,565 23.4% Studies using non-numerical data collection and interpretive analytical approaches, including interviews, focus groups, ethnography, content analysis of textual or visual data, and observational methods.
mixed_methods 2,172 9.1% Studies that explicitly integrate both quantitative and qualitative data collection and analysis methods. Classified only when the abstract explicitly describes integrating both approaches, not when multiple methods of the same type are used.
review 1,274 5.4% Systematic reviews, scoping reviews, meta-analyses, and other review studies that synthesize existing literature. Includes all review subtypes.
other 240 1.0% Presentations not fitting other categories, including conceptual and theoretical papers, program descriptions, pedagogical presentations, policy analyses without primary data, methodological discussions, implementation case studies, and historical analyses.

Classification Process

Each abstract was classified by the gpt-oss-20b language model, which analyzed the text for methodological indicators including data type descriptions, sample characteristics, analytical methods, and explicit methodology statements. The model’s classification rationale is preserved in the methodology_classification_rationale field.

When the abstract mentioned multiple approaches, classification was based on the dominant methodology unless the abstract explicitly described integrating quantitative and qualitative components, which triggered a mixed_methods classification.

Academic Position Taxonomy

Author positions were extracted from Confex affiliation text by the gpt-oss-20b model, then normalized into 16 standardized categories using keyword-based classification with regular expression pattern matching. Position indicators were searched in priority order to handle records containing multiple position keywords.

Normalization Procedures

Author Names

Author names were normalized using NFKD (Normalization Form Compatibility Decomposition) from Python’s unicodedata module. This process decomposes characters into base forms, removes combining marks, and converts accented characters to ASCII equivalents. Whitespace was collapsed and asterisks marking presenting authors were removed. The resulting normalized names are stored in name_normalized while preserving original casing in the name field.

Institutions

Institution names were normalized through a three-stage matching process: exact matching against canonical forms, substring matching for variants with department prefixes or suffixes, and pattern matching for common naming variations. A language model assisted in creating the mapping table of canonical institution names and their associated variants. The normalization pipeline first cleaned institution strings by removing embedded position titles and geographic suffixes.

Countries/Regions

Country/region names were standardized using a mapping dictionary addressing over 80 name variations. Assignment followed explicit mention in the affiliation text or inference from state/city information. State and province abbreviations within the United States and Canada were expanded to full names.

Entity Resolution

Unique researchers were identified using a graph-based entity resolution system combining fuzzy string similarity (RapidFuzz library), blocking by first initial and last name, and additional factors including matching middle initials, shared institutions, and logical career-stage progression. Name pairs with similarity scores of 90 or higher were connected in a NetworkX graph, and connected components identified author clusters. This process reduced 23,481 raw name variants to 20,779 canonical identities.
