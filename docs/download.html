Recommended: Use the MCP connection to query the database directly from your AI assistant. The data is hosted on Supabase with virtually no latency — no downloads, no file management, no version mismatches. You get real-time access to the full dataset including vector embeddings for semantic search.

Running locally? We recommend cloning the repository and using an agentic tool like Claude Code or OpenCode with the included skill files (CLAUDE.md, AGENTS.md). These files give the AI complete schema context, query patterns, data quality notes, and semantic search instructions — resulting in faster, more accurate analysis than writing code manually.

MCP Connection (Cloud)

Add this to your MCP configuration to query the live database with virtually no latency. No API key or Node.js required.

{
  "mcpServers": {
    "sswr-history": {
      "url": "https://mcp.supabase.com/mcp?project_ref=jomsksqqcpkbuhwytovo&read_only=true&features=database"
    }
  }
}

Works with Claude Desktop, Claude Code, Cursor, OpenCode, and any MCP-compatible client. See the Quick Start for step-by-step setup.

Browse Files

Vector embeddings are available through the REST API and MCP connection only. They are not included in the downloadable files due to their size. Embeddings are intended for semantic similarity search against the database, not for direct download. See the embeddings section below for usage.

REST API

The database is also accessible via the Supabase PostgREST API for programmatic access. The sswr-query skill handles this automatically — just ask your AI assistant to query the API. Connection details are in the CODEBOOK_LLM.md portable reference.

Vector Embeddings

Pre-computed vector embeddings are available for all 23,793 presentations, enabling semantic similarity search — finding presentations by meaning, not just keywords.

ModelDimensionsBest For
text-embedding-3-large3,072Higher precision search and clustering
text-embedding-3-small1,536General similarity search (faster, cheaper)

Embeddings are not available for direct download. To run a similarity search, use the sswr-semantic-search skill — it handles the entire workflow including embedding generation via OpenRouter and the Supabase RPC call. Install the skills via the Quick Start.

Example: “Find presentations similar to ‘racial disparities in mental health treatment’”
