Quick Start

Why AI-first?

This database is designed to be queried through conversation, not code. We are not encouraging researchers to hand off analysis to AI. The opposite: we want to free you from boilerplate so you can focus entirely on thinking about your research questions, interpreting results, and understanding the data.

Writing pd.read_csv() or library(readr) is not a scholarly contribution. It’s plumbing. The intellectual work is deciding what to ask, how to interpret, and what it means. AI handles the plumbing. You do the thinking.

Step 1 — Connect to the Database

Give your AI assistant access to the live SSWR database via MCP. Copy this prompt and paste it into Claude Desktop, Claude Code, Cursor, or OpenCode:

Preview
I want to connect to the SSWR Conference History Database via MCP. The server name is “sswr-history” and the URL is https://mcp.supabase.com/mcp?project_ref=jomsksqqcpkbuhwytovo&read_only=true&features=database. Please add this to my MCP configuration, help me restart, and verify it works by listing the available tables.

Already know MCP? The config details are on the Download page.

Step 2 — Load the Skill File

Skill files give the AI complete context about the database — schema, query patterns, categorical values, and semantic search instructions. Copy this prompt and give it to the same AI assistant:

Preview
Please download the SSWR skill file from https://github.com/beperron/SSWR-History-Public/raw/main/CODEBOOK_LLM.md and read it. This contains the complete database schema, query patterns, categorical values, data quality notes, and semantic search instructions. Confirm you have the context loaded and summarize what tables are available.

Step 3 — Start Asking Questions

You’re set up. Try these:

“How many presentations were at SSWR in each year from 2005 to 2026?”

“What are the top 20 institutions by first-authored presentations?”

“How has the proportion of qualitative vs. quantitative research changed over time?”

“How has international participation at SSWR changed since 2010?”

“Find presentations about substance use interventions from the last 5 years. Who are the most active researchers?”

“Compare the methodology distribution of University of Michigan vs. Columbia University.”

Prefer files?

The data is also available as SQLite, CSV, and JSON downloads, plus a REST API.
