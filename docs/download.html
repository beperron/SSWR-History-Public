Download the Data

The SSWR Conference History Database is available in CSV and JSON formats. All files are encoded in UTF-8. See the Codebook for complete field documentation.

License: This dataset is released under Creative Commons Attribution 4.0 (CC BY 4.0). You are free to share and adapt the data for any purpose, provided you give appropriate credit. See How to Cite.

CSV Files (Recommended)

Best for use with R, Python (pandas), Stata, SPSS, or spreadsheet software.

papers.csv

23,793 presentations with titles, abstracts, and methodology classifications. All 8 fields from the papers table.

paper_authors.csv

69,924 author–presentation records with names, affiliations, positions, institutions, and geographic information. All 21 fields from the paper_authors table.

Quick Start

Python

import pandas as pd

# Load the data
papers = pd.read_csv('papers.csv')
authors = pd.read_csv('paper_authors.csv')

# Join tables
merged = authors.merge(papers, left_on='paper_id', right_on='id', suffixes=('_author', '_paper'))

# Example: methodology distribution
print(papers['methodology'].value_counts())

# Example: presentations per year
print(papers.groupby('year').size())

R

library(readr)

# Load the data
papers <- read_csv("papers.csv")
authors <- read_csv("paper_authors.csv")

# Join tables
merged <- merge(authors, papers, by.x = "paper_id", by.y = "id")

# Example: methodology distribution
table(papers$methodology)

# Example: top institutions
sort(table(authors$institution_normalized), decreasing = TRUE)[1:20]
